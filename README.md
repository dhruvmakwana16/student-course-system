# Student Course System

A full-stack MERN application for managing courses and student learning.

## Features

* User authentication (JWT)
* Course browsing and enrollment
* Progress tracking
* Admin dashboard (add/update/delete courses)
* Real-time chat using Socket.io

## Tech Stack

* Frontend: React, Tailwind CSS
* Backend: Node.js, Express.js
* Database: MongoDB

## Setup

```bash
git clone https://github.com/YOUR_USERNAME/student-course-system.git
cd student-course-system
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
